Welcome!

This application is my playground for exploring features in Bolero, Bulma, and possibly other tools that I use to build my business applications. By keeping these explorations in a public repo, I hope to give some value back to the community and the creators of Bolero, Bulma and other great tools.

The application was originally developed from this template - https://github.com/Tarmil/bolero.template.structured - and then I refactored and adjusted it to my liking before going on to explore more Bulma stuff.

The original Counter and Books demo pages, and the following bullet points from the original Home page, has been kept. They work the same in the Bolero default template and in the structured template, but as the name somewhat hints at, the structured template takes advantage of Elmish submodels. There are also many other differences that are important when developing applications that grow.

  • The entire application is driven by Elmish.
  • The menu on the left switches pages based on routes.
  • The Counter page demonstrates event handlers and data binding in HTML templates.
  • The Download data page demonstrates the use of remote functions.

Several more Bulma demos are in the workings.
